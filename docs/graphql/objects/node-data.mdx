```graphql
type NodeData {
  stakedTotal: String!
  totalNodes: Int!
  inactiveNodes: Int!
  validatingNodes: Int!
  uptime: Float!
}
```

### Fields

#### `stakedTotal` ([`String!`](/docs/graphql/scalars/string))

Total staked amount across all nodes

#### `totalNodes` ([`Int!`](/docs/graphql/scalars/int))

Total number of nodes

#### `inactiveNodes` ([`Int!`](/docs/graphql/scalars/int))

Number of inactive nodes

#### `validatingNodes` ([`Int!`](/docs/graphql/scalars/int))

Number of nodes validating

#### `uptime` ([`Float!`](/docs/graphql/scalars/float))

Total uptime for all epochs across all nodes. Or specify a number of epochs
