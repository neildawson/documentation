Reward information for a single party

```graphql
type Reward {
  assetId: String!
  partyId: String!
  epoch: Int!
  amount: String!
  percentageOfTotal: String!
  receivedAt: String!
}
```

### Fields

#### `assetId` ([`String!`](/docs/graphql/scalars/string))

The asset for which this reward is associated

#### `partyId` ([`String!`](/docs/graphql/scalars/string))

Party receiving the reward

#### `epoch` ([`Int!`](/docs/graphql/scalars/int))

Epoch for which this reward was distributed

#### `amount` ([`String!`](/docs/graphql/scalars/string))

Amount received for this reward

#### `percentageOfTotal` ([`String!`](/docs/graphql/scalars/string))

Percentage out of the total distributed reward

#### `receivedAt` ([`String!`](/docs/graphql/scalars/string))

Time at which the rewards was received
