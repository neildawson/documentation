No description

```graphql
type FutureProduct {
  maturity: String!
  settlementAsset: Asset!
  quoteName: String!
  oracleSpecForSettlementPrice: OracleSpecConfiguration!
  oracleSpecForTradingTermination: OracleSpecConfiguration!
  oracleSpecBinding: OracleSpecToFutureBinding!
}
```


### Fields

#### `maturity` ([`String`](/docs/v0.47/graphql/scalars/string))

RFC3339Nano time when the future products matures


#### `settlementAsset` ([`Asset`](/docs/v0.47/graphql/objects/asset))

Product asset ID


#### `quoteName` ([`String`](/docs/v0.47/graphql/scalars/string))

String representing the quote (e.g. BTCUSD -&#x003E; USD is quote)


#### `oracleSpecForSettlementPrice` ([`OracleSpecConfiguration`](/docs/v0.47/graphql/objects/oracle-spec-configuration))

Describes the oracle data that an instrument wants to get from the oracle engine for settlement price.


#### `oracleSpecForTradingTermination` ([`OracleSpecConfiguration`](/docs/v0.47/graphql/objects/oracle-spec-configuration))

Describes the oracle data that an instrument wants to get from the oracle engine for trading termination.


#### `oracleSpecBinding` ([`OracleSpecToFutureBinding`](/docs/v0.47/graphql/objects/oracle-spec-to-future-binding))

OracleSpecToFutureBinding tells on which property oracle data should be
used as settlement price.
