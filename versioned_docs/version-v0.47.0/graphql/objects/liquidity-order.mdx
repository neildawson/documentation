A special order type for liquidity providers

```graphql
type LiquidityOrder {
  reference: PeggedReference!
  proportion: Int!
  offset: Int!
}
```


### Fields

#### `reference` ([`PeggedReference`](/docs/v0.47/graphql/enums/pegged-reference))

The value to which this order is tied


#### `proportion` ([`Int`](/docs/v0.47/graphql/scalars/int))

The proportion of the commitment allocted to this order


#### `offset` ([`Int`](/docs/v0.47/graphql/scalars/int))

Offset from the pegged reference
