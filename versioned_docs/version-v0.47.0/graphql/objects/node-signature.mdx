title: NodeSignature
---


Represents a signature for the approval of a resource from a validator

```graphql
type NodeSignature {
  id: ID!
  signature: String
  kind: NodeSignatureKind
}
```


### Fields

#### `id` ([`ID`](/docs/v0.47/graphql/scalars/id))

The id of the resource being signed for


#### `signature` ([`String`](/docs/v0.47/graphql/scalars/string))

The signature, as base64 encoding


#### `kind` ([`NodeSignatureKind`](/docs/v0.47/graphql/enums/node-signature-kind))

The kind of signature this is (e.g: withdrawal, new asset, etc)
