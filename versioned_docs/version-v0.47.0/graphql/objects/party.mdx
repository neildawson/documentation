Represents a party on Vega, could be an ethereum wallet address in the future

```graphql
type Party {
  id: ID!
  orders(
  skip: Int
  first: Int
  last: Int
): [Order!]
  trades(
  marketId: ID
  skip: Int
  first: Int
  last: Int
): [Trade!]
  accounts(
  marketId: ID
  asset: String
  type: AccountType
): [Account!]
  positions: [Position!]
  margins(
  marketId: ID
): [MarginLevels!]
  proposals(
  inState: ProposalState
): [Proposal]
  votes: [ProposalVote]
  withdrawals: [Withdrawal!]
  deposits: [Deposit!]
  liquidityProvisions(
  market: String
  reference: String
): [LiquidityProvision!]
  delegations(
  nodeId: String
): [Delegation!]
  stake: PartyStake!
  rewardDetails: [RewardPerAssetDetail]
}
```
