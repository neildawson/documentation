---


Create an order linked to an index rather than a price

```graphql
type PeggedOrder {
  reference: PeggedReference!
  offset: String!
}
```


### Fields

#### `reference` ([`PeggedReference`](/docs/v0.47/graphql/enums/pegged-reference))

Index to link this order to


#### `offset` ([`String`](/docs/v0.47/graphql/scalars/string))

Price offset from the peg
