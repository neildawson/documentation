The details of a withdrawal processed by vega

```graphql
type Withdrawal {
  id: ID!
  party: Party!
  amount: String!
  asset: Asset!
  status: WithdrawalStatus!
  ref: String!
  expiry: String!
  createdTimestamp: String!
  withdrawnTimestamp: String
  txHash: String
  details: WithdrawalDetails
}
```


### Fields

#### `id` ([`ID`](/docs/v0.47/graphql/scalars/id))

The Vega internal id of the withdrawal


#### `party` ([`Party`](/docs/v0.47/graphql/objects/party))

The PartyID initiating the witndrawal


#### `amount` ([`String`](/docs/v0.47/graphql/scalars/string))

The amount to be withdrawn


#### `asset` ([`Asset`](/docs/v0.47/graphql/objects/asset))

The asset to be withdrawn


#### `status` ([`WithdrawalStatus`](/docs/v0.47/graphql/enums/withdrawal-status))

The current status of the withdrawal


#### `ref` ([`String`](/docs/v0.47/graphql/scalars/string))

A reference the foreign chain can use to refere to when processing the withdrawal


#### `expiry` ([`String`](/docs/v0.47/graphql/scalars/string))

RFC3339Nano time until the withdrawal will be invalid


#### `createdTimestamp` ([`String`](/docs/v0.47/graphql/scalars/string))

RFC3339Nano time at which the withdrawal was created


#### `withdrawnTimestamp` ([`String`](/docs/v0.47/graphql/scalars/string))

RFC3339Nano time at which the withdrawal was finalized


#### `txHash` ([`String`](/docs/v0.47/graphql/scalars/string))

Hash of the transaction on the foreign chain


#### `details` ([`WithdrawalDetails`](/docs/v0.47/graphql/unions/withdrawal-details))

Foreign chain specific details about the withdrawal
