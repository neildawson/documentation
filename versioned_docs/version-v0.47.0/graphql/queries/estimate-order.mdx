---
id: estimate-order
title: estimateOrder
---


return an estimation of the potential cost for a new order

```graphql
estimateOrder(
  marketId: ID!
  partyId: ID!
  price: String
  size: String!
  side: Side!
  timeInForce: OrderTimeInForce!
  expiration: String
  type: OrderType!
): OrderEstimate!
```


### Arguments

#### `marketId` ([`ID`](/docs/v0.47/graphql/scalars/id))

ID of the market to place the order


#### `partyId` ([`ID`](/docs/v0.47/graphql/scalars/id))

ID of the party placing the order


#### `price` ([`String`](/docs/v0.47/graphql/scalars/string))

Price of the asset


#### `size` ([`String`](/docs/v0.47/graphql/scalars/string))

Size of the order


#### `side` ([`Side`](/docs/v0.47/graphql/enums/side))

Side of the order (Buy or Sell)


#### `timeInForce` ([`OrderTimeInForce`](/docs/v0.47/graphql/enums/order-time-in-force))

TimeInForce of the order


#### `expiration` ([`String`](/docs/v0.47/graphql/scalars/string))

expiration of the the order


#### `type` ([`OrderType`](/docs/v0.47/graphql/enums/order-type))

type of the order


### Type

#### [`OrderEstimate`](/docs/v0.47/graphql/objects/order-estimate) 

An estimate of the fee to be paid by the order
