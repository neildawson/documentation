Return a list of aggregated node signature for a given resource ID

```graphql
nodeSignatures(
  resourceId: ID!
): [NodeSignature!]
```


### Arguments

#### `resourceId` ([`ID`](/docs/v0.47/graphql/scalars/id))




### Type

#### [`NodeSignature`](/docs/v0.47/graphql/objects/node-signature) 

Represents a signature for the approval of a resource from a validator
