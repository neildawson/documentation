Subscribe to delegation data

```graphql
delegations(
  party: ID
  nodeID: ID
): Delegation!
```


### Arguments

#### `party` ([`ID`](/docs/v0.47/graphql/scalars/id))

the party to subscribe for, empty if all


#### `nodeID` ([`ID`](/docs/v0.47/graphql/scalars/id))

the node to subscribe for, empty if all


### Type

#### [`Delegation`](/docs/v0.47/graphql/objects/delegation)
