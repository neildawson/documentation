Subscribe to the mark price changes

```graphql
marketData(
  marketId: ID
): MarketData!
```


### Arguments

#### `marketId` ([`ID`](/docs/v0.47/graphql/scalars/id))

id of the market we want to subscribe to the market data changes


### Type

#### [`MarketData`](/docs/v0.47/graphql/objects/market-data) 

Live data of a Market
